# Autocomplete

React autocomplete component

## How to run
1. Clone this repository
  ```bash
  $> git clone https://github.com/RusPosevkin/Autocomplete.git
  ```

2. Install dependencies
  ```bash
  $> cd ./Autocomplete
  $> npm i
  ```

3. Run app
  ```bash
  $> npm start
  ```

4. Open app in a browser at [http://localhost:3000](http://localhost:3000).

5. A few tests
  ```bash
  $> npm test
  ```
